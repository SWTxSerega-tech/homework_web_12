Contacts REST API

REST API для зберігання та управління контактами. Застосунок побудовано на фреймворку FastAPI з використанням SQLAlchemy (синхронний режим) та PostgreSQL. Додатково реалізовано повноцінну автентифікацію та авторизацію користувачів за допомогою JWT токенів (Access та Refresh токени).

Основні папки программи:

- auth.py - Модуль безпеки, який відповідає за генерацію JWT токенів доступу та оновлення, хешування паролів за допомогою bcrypt та перевірку поточного авторизованого користувача.

- crud.py - Набір синхронних функцій для взаємодії з базою даних, що реалізують логіку додавання користувачів, ізольованого керування контактами та пошуку за іменинниками чи текстовими запитами.

- database.py - Файл, який за допомогою python-dotenv зчитує захищені змінні оточення, створює двигун SQLAlchemy та функцію-генератор для підключення до сесії бази даних.

- main.py - Точка входу в застосунок, де ініціалізується FastAPI, автоматично створюються таблиці при запуску та описані всі захищені авторизацією маршрути (endpoints) для роботи з API.

- models.py - Опис структури таблиць у базі даних (користувачі та контакти) із налаштуванням зв'язків «один до багатьох» та каскадного видалення через ORM SQLAlchemy.

- schemas.py - Схеми валідації Pydantic, що перевіряють типи даних на відповідність (наприклад, формат EmailStr чи довжину рядків) для вхідних запитів та вихідних відповідей API.

Основні можливості

- Керування користувачами: Реєстрація (SignUp) з хешуванням паролів та Авторизація (Login) з видачею пари JWT токенів.
- Безпека: Кожен користувач має ізольований доступ і може переглядати, створювати чи редагувати виключно свої контакти.
- Повний CRUD для контактів: Створення, читання, оновлення та видалення записів.
- Розширений пошук: Пошук контактів за іменем, прізвищем або електронною поштою (Query параметри).
Календар днів народжень: Отримання списку контактів, у яких день народження припадає на найближчі 7 днів.


 Вимоги до середовища

- Python 3.11 або новіша
- Docker (для підняття локальної бази даних)
- Poetry (для керування залежностями та віртуальним середовищем)

 Встановлення залежностей

У цьому проєкті для керування пакетами використовується Poetry.

1. Клонуйте репозиторій та перейдіть до кореневої папки проєкту:
   ```bash
   cd homework_web_12

2. Налаштуйте створення віртуального середовища проєкту:
    poetry config virtualenvs.in-project true

3. Встановіть усі залежності:
    poetry install
